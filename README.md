# Beginner Python Projects

A collection of beginner-level Python projects I am building as I learn programming and move towards my goal of becoming an AI developer.

## 1. Python Console Calculator

A simple command-line calculator that performs basic arithmetic operations on two numbers and runs in a loop until the user exits.

**Features:**

- Addition, subtraction, multiplication, division
- Input validation for non-numeric values
- Handles division by zero
- Menu-based interface

**What I practised:**

- `input()` and number conversion
- Functions
- `if / elif / else` and loops
- Basic error handling

**Code:**  
`calculator.py` in the `Python-calculator` repository
